## 6-vazifa (Windows): Messenjer token va sessiyalarni aniqlash (1200 ball)

Windows tizimida (jonli yoki disk image) messenjer tokenlar, sessiya cookie’lar va autentifikatsiya ma’lumotlarini aniqlash — forensik tahlilning muhim qismi. Bu ma’lumotlar buzg‘unchining qaysi xizmatlarga kirganligini, qanday ma’lumotlarni o‘g‘irlaganligini va qanday hisoblarni kompromatatsiya qilganligini ko‘rsatadi.


### 1. QAYERDAN QIDIRISH KERAK? (STORAGE LOCATIONS)

**Browser cookie va tokenlar:**

| Brauzer | Cookie fayl joylashuvi | Token/lokal storage |
|---------|----------------------|---------------------|
| **Chrome/Edge** | `%LocalAppData%\Google\Chrome\User Data\Default\Cookies` | `%LocalAppData%\Google\Chrome\User Data\Default\Local Storage\leveldb`  |
| **Firefox** | `%AppData%\Mozilla\Firefox\Profiles\[profile]\cookies.sqlite` | `%AppData%\Mozilla\Firefox\Profiles\[profile]\webappsstore.sqlite`  |
| **Brave** | `%LocalAppData%\BraveSoftware\Brave-Browser\User Data\Default\Cookies` | Shunga o‘xshash chromium-based  |

**Messenjer tokenlar:**

| Messenjer | Token/sessiya joylashuvi |
|-----------|-------------------------|
| **Telegram** | `%AppData%\Telegram Desktop\tdata\*_key`, `*_data` fayllar (encrypted) |
| **Discord** | `%AppData%\discord\Local Storage\leveldb`; `%AppData%\discord\Cookies` |
| **WhatsApp (Web)** | Brauzer cookie/lokal storage orqali |
| **Signal (Desktop)** | `%AppData%\Signal\sql\db.sqlite` (encrypted) |
| **Slack/Teams** | Brauzer cookie yoki desktop app lokal storage |

**Windows sistemasi credentiallari:**

| Credential turi | Joylashuvi | Izoh |
|----------------|-----------|------|
| **Windows Credentials** | `%UserProfile%\AppData\Roaming\Microsoft\Credentials\` | DPAPI bilan shifrlangan  |
| **Windows Vault** | `%UserProfile%\AppData\Local\Microsoft\Vault\` | IE/Edge parollari, Windows Mail  |
| **DPAPI master key** | `%UserProfile%\AppData\Roaming\Microsoft\Protect\` | SID bilan nomlangan  |

## 3. Telegram Tdata Decrypt (Telegram token va sessiyalar)

Telegram lokal ma’lumotlari `tdata` papkasida saqlanadi. Buni decrypt qilish orqali boshqa tizimda Telegram’ga kirmasdan sessionni tiklash mumkin.

### 3.1. Telegram ma’lumotlari qayerda

```
%AppData%\Telegram Desktop\tdata\
```
Muhim fayllar:
- `key_datas` — asosiy decrypt kaliti (localKey)
- `D877F783D5D3EF8Cs` — birinchi akkaunt uchun session
- `D877F783D5D3EF8C\map` — foydalanuvchi ma’lumotlari (phone, username)

Agar bir necha akkaunt bo‘lsa: `data#2` → `A7FDF864FBC10B77s` va `A7FDF864FBC10B77\map`

### 3.2. Telegram sessionni boshqa tizimga ko‘chirish (account hijacking)

**Eng oddiy usul — fayllarni nusxalash:**
```cmd
# Manba tizimdan (image yoki live)
copy "%AppData%\Telegram Desktop\tdata\key_datas" "D:\extracted\"
copy "%AppData%\Telegram Desktop\tdata\D877F783D5D3EF8Cs" "D:\extracted\"
copy "%AppData%\Telegram Desktop\tdata\D877F783D5D3EF8C\map" "D:\extracted\"
```

**Yangi tizimda qayta tiklash:**
1. Telegram Desktop’ni yoping
2. `%AppData%\Telegram Desktop\tdata\` papkasini tozalang
3. Nusxalangan fayllarni joylashtiring
4. Telegram’ni oching — avtomatik login bo‘ladi

### 3.3. Telegram ma’lumotlarini decrypt qilish (manual tahlil)

Agar tdata fayllari bor bo‘lsa, lekin ularni ochish kerak bo‘lsa:

```bash
# Telegram Desktop Decrypt tool
git clone https://github.com/atilaromero/telegram-desktop-decrypt.git
cd telegram-desktop-decrypt

```

Bu skript:
- `key_datas` dan localKey ni chiqaradi
- Har bir akkaunt uchun phone number, username, user_id ni ko‘rsatadi
- Session kalitlarini eksport qiladi

Agar siz disk image bilan ishlayotgan bo‘lsangiz, Autopsy GUI orqali brauzer ma’lumotlarini olish eng qulay.

**O‘rnatish:**
```bash
sudo apt update && sudo apt install autopsy -y
autopsy
# Browserda http://127.0.0.1:9999 ochiladi
```

**Ishlatish:**
1. `Create New Case` – case nomi va papkasini belgilang
2. `Add Data Source` → `Disk Image or VM File` – image faylni tanlang
3. `Next` → `Next` → `Finish` – image tahlili boshlanadi
4. `Web History` tabiga o‘ting – Chrome, Firefox, Edge history va cookie’lari ko‘rinadi
5. `Deleted Files` – o‘chirilgan history fayllarini tiklash mumkin

**Eksport:** `Case` → `Export` → CSV yoki HTML formatda hisobot.

## 5. QAYERDAN QIDIRISH KERAK? (FAYLAR JOYLASHUVI)

| Ma’lumot turi | Joylashuvi (Linux) |
|--------------|-------------------|
| **Chrome cookie** | `~/.config/google-chrome/Default/Cookies` (SQLite) |
| **Chrome localStorage** | `~/.config/google-chrome/Default/Local Storage/leveldb/` |
| **Firefox cookie** | `~/.mozilla/firefox/*.default/cookies.sqlite` |
| **Firefox localStorage** | `~/.mozilla/firefox/*.default/webappsstore.sqlite` |
| **Brave cookie** | `~/.config/BraveSoftware/Brave-Browser/Default/Cookies` |
| **Telegram tdata** | `~/.local/share/TelegramDesktop/tdata/` |
| **Discord** | `~/.config/discord/Local Storage/leveldb/` |

**Disk image mount qilinganda:** `/mnt/linux` prefiksi bilan yuqoridagi yo‘llar.

## 6. MAKSIMAL NATIJA UCHUN STRATEGIYA

| Bosqich | Harakat | Instrument | Vaqt |
|--------|---------|------------|------|
| 1 | Image mount (agar image bo‘lsa) | `mount`, `ewfmount` | 5 min |
| 2 | Brauzer cookie va localStorage | HackBrowserData | 5 min |
| 3 | Telegram tdata decrypt | Telegram Decryptor v2.0 | 5 min |
| 4 | Windows credential (agar image Windows bo‘lsa) | dploot | 10 min |
| 5 | Autopsy bilan chuqur tahlil (ixtiyoriy) | autopsy | 10 min |
| 6 | Hisobot tayyorlash | CSV/JSON eksport | 5 min |

**Hisobotga yoziladigan ma’lumotlar:**
- Brauzer: domain, cookie name, token value (hash), expiry time
- Telegram: user_id, dc_id, auth_keys
- Windows credential: target server, username

## 7. QISQA MISOL (LIVE LINUX)

```bash
# 1. HackBrowserData yuklash
git clone https://github.com/moonD4rk/HackBrowserData
cd HackBrowserData/cmd/hack-browser-data
go build

# 2. Barcha brauzer ma’lumotlarini olish
./hack-browser-data -b all -f json --dir ~/forensics_output

# 3. Telegram tdata decrypt
git clone https://github.com/Lucky7897/-Telegram-Desktop-TData-Decryptor-v2.0
cd -Telegram-Desktop-TData-Decryptor-v2.0
pip install -r requirements.txt
python telegram_decryptor.py
# GUI da "Auto Search" → "Decrypt"

# 4. Natijalarni tekshirish
cat ~/forensics_output/*.json
cat ~/forensics_output/telegram_decrypt.json
```
